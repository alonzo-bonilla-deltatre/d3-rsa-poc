# SearchBarOverlay

Use the _SearchBarOverlay_ component to show the SearchBar component within the Overlay component. An HTML element
(e.g.: button) with data-tag=search is required to trigger the toggle onto the SearchBarOverlay

### Default

The default version with black background and nothing related to results
